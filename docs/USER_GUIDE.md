# RakshaCircle User Guide

## What Users Do

1. Connect a Freighter wallet and verify the Stellar Testnet network.
2. Create a profile with your name.
3. Add trusted contacts to your safety circle.
4. Trigger SOS events with location hints and context.
5. Acknowledge events from a trusted wallet.
6. Review the production dashboard for metrics, monitoring, and indexed events.

## User Onboarding and Feedback Collection

Use a Google Form to capture these fields for every verified tester:

- Wallet address
- Email address
- Name
- Product feedback rating
- Freeform feedback comment

Export the form responses to Excel for analysis and keep the sheet linked in the README.

## How to Use Fee Sponsorship

The SOS form includes a fee sponsorship toggle. When enabled, the backend prepares a sponsored transaction preview so the flow can be upgraded to gasless execution.

## Production Dashboard

The production-readiness panel shows:

- Verified user counts
- DAU and 30-day retention
- Monitoring and uptime signals
- Indexed SOS events and search endpoints
- Security checklist coverage
